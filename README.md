### Built With

- ExpressJS
- Sequelize
- Postgres
- Swagger Open API

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AldiAgeng/car-management-api.git
   ```
2. Install packages
   ```sh
   yarn install
   ```
3. Next Step

   ```sh
   - setting file database.js
   - sequelize db:create
   - sequelize db:migrate
   - sequelize db:seed:all
   ```

4. Run

   ```sh
   yarn start
   ```
